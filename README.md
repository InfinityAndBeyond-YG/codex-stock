# Stock Flow

계좌별, 종목별 주식 흐름을 한 화면에서 보고, 월별 자산 변화와 스크린샷 OCR 입력까지 다룰 수 있는 반응형 포트폴리오 대시보드입니다.

## 포함한 기능

- 모바일과 데스크톱에 맞춘 반응형 화이트 톤 대시보드
- 총 평가 자산, 주식 평가 금액, 손익, 전월 대비 흐름 요약
- 계좌별 보기 / 종목별 보기 전환
- 선택한 계좌 또는 종목의 상세 정보 편집
- 최근 6개월 기준 월별 계좌 자산 흐름 막대 그래프
- 종목별 월별 매입금 대비 평가금 비교 그래프
- 은행·증권사 스크린샷 업로드 후 OCR 텍스트 인식
- OCR 결과에서 보유 수량, 평균 단가, 매수/매도 거래 자동 분류 후 반영
- 브라우저 `localStorage` 저장

## 사용 방법

1. 첫 화면에서 데모 데이터로 계좌/종목 흐름을 확인합니다.
2. `보기 기준`을 `종목` 또는 `계좌`로 바꿔 세부 화면을 탐색합니다.
3. 상세 패널에서 종목 수량, 평단, 현재가, 계좌 현금을 수정하면 즉시 자산 요약과 차트가 다시 계산됩니다.
4. 하단 `스크린샷 업로드` 영역에 은행 또는 증권사 캡처 이미지를 올립니다.
5. OCR 결과를 확인한 뒤 `인식 결과 반영`을 누르면 보유 종목/거래 내역이 저장됩니다.
6. OCR이 애매할 경우 `수동 거래 입력`으로 한 건씩 추가할 수 있습니다.

## OCR 참고

- OCR은 브라우저에서 `Tesseract.js`를 CDN으로 불러와 처리합니다.
- 처음 OCR을 실행할 때는 언어 데이터 로딩 때문에 인터넷 연결이 필요할 수 있습니다.
- 인식 정확도는 스크린샷 해상도와 화면 구성에 따라 달라집니다.
- 현재는 한국어/영문 텍스트에서 아래 같은 패턴을 우선적으로 인식합니다.

예시:

- `삼성전자 12주 평단 71000 현재가 74800`
- `2026-03-20 NAVER 매수 2주 210000`
- `03/18 카카오 매도 5주 46800`

## 현재 구현 범위

- 자산 합계와 차트는 앱 안에 저장된 데이터가 바뀌는 즉시 다시 계산됩니다.
- 실시간 증권사 API 또는 실시간 시세 스트리밍 연결은 아직 붙여두지 않았습니다.
- 지금 버전은 `수동 현재가 입력`과 `스크린샷 반영` 중심으로 빠르게 확인하는 구조입니다.
